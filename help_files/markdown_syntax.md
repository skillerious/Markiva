**Markdown** is a lightweight markup language for creating formatted text using a plain-text editor.

## üìù 1. Headings

Use `#` for headings. The number of `#` symbols determines the level of the heading.

- `# Heading 1`
- `## Heading 2`
- `### Heading 3`
- `#### Heading 4`
- `##### Heading 5`
- `###### Heading 6`
